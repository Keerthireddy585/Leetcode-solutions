# SQL Practice Repository

This repository contains a comprehensive collection of SQL problem solutions implemented using MySQL-style syntax. The problems are primarily sourced from real-world data scenarios and coding platforms (such as LeetCode) and focus on building strong fundamentals in data querying, analysis, and transformation.
The goal of this repository is to demonstrate hands-on SQL proficiency, covering beginner to intermediate concepts commonly required for Data Analyst, Business Analyst, and Software roles.

Through these SQL queries, the following concepts are extensively practiced:

- `SELECT`, `WHERE`, `ORDER BY`
- `GROUP BY`, `HAVING`
- `INNER JOIN`, `LEFT JOIN`
- Subqueries & Nested Queries
- Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Conditional Logic (`CASE WHEN`)
- Window Functions (`RANK`, `DENSE_RANK`)
- Data Cleaning & Validation
- Date & Time Functions
- Handling `NULL` values
- Duplicate detection & removal
